 Mood Tracker 

A C++ program to track weekly moods and provide a summary report. Designed to demonstrate OOP concepts like classes, private/public members, arrays, and functions.

Features:

Input daily moods for a full week.

Recognizes moods: happy, sad, sleepy, crying, excited, and others.

Displays a daily mood report with personalized messages.

Provides a summary highlighting the most frequent moods.

How to Use

Clone or download the repository.

Open Project1.cpp in any C++ IDE.

Run the program and enter moods for each day.

View the daily report and weekly summary.
